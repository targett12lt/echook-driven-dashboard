eChook dweet data - demo page


This is a very basic example web interface using Vue.js to fetch and show dweet.io data from an eChook, hosted for free on Github! If you want to create your own dashbard, feel free to fork this repository and make the site your own.

For more information please go to the eChook documentation. TODO: Write the documentation for this and update the link!

Note: dweet.io is SLOW! It's slowed down considerably since eChook started using it. This site by default polls for new data every 5 seconds, but it will still occasionally rate limit.

Enter your 'thing name' and hit go!
